Demo of web-component-decorator

Below is a simple button defined as a web component using the @define and @attribute decorators. This is the HTML code of the button:

<my-button id="btn" icon="favorite">button</my-button>

You can set the icon of the button by changing the icon attribute. You can do this by clicking the 'do' buttons below.

button
Set icon attribute 
document.getElementById('btn').setAttribute('icon', 'build')

document.getElementById('btn').setAttribute('icon', 'favorite')

document.getElementById('btn').setAttribute('icon', 'visibility')

Below you can find the source code of the button web component, which can be found in demo/my-button.ts. 

import { attribute, CustomElement, define } from "web-component-decorator";

@define("my-button")
class MyButton extends HTMLElement implements CustomElement {
  constructor() {
    super();

    this.attachShadow({ mode: "open" });

    this.shadowRoot.innerHTML = `
          <style>
            button {
              padding: 5px;
            }
            #icon {
              vertical-align: middle;
            }
          </style>

          <button id="button">
              <img id="icon" alt="" width="16" height="16">          
              <slot></slot>
          </button>
        `;
  }

  @attribute("icon")
  setIcon(icon: string, oldIcon: string) {
    this.shadowRoot.getElementById("icon").setAttribute('src', `icons/${icon}-24px.svg`);
  }
}
